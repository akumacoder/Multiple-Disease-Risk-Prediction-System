# Multiple-Disease-Risk-Prediction-System

The Multi-Disease Prediction System is an AI/ML-based application designed to predict the likelihood of Heart Disease, Diabetes, and Kidney Disease using patient health data.It focuses on early detection by applying machine learning models optimized for high recall, minimizing false negatives. The system uses data preprocessing, feature engineering, and model evaluation techniques to ensure reliable predictions. Multiple algorithms such as Logistic Regression, Random Forest, and SVM are trained and compared for performance. The final model is integrated into an interactive web interface for real-time predictions. The application is built using Python and deployed using Streamlit. This project demonstrates an end-to-end machine learning pipeline from data analysis to deployment.

Features:
1. Predicts Heart Disease, Diabetes, and Kidney Disease
2. Uses AI/ML models for accurate risk prediction
3. Handles imbalanced data using SMOTE
4. Interactive Streamlit web interface
5. Real-time prediction based on user input
6. Displays risk levels (Low, Medium, High)
7. Model selection based on performance metrics (Recall, Precision, F1-score)
